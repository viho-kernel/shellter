## 📞 IF YOU GET STUCK

### Step 1: Check Your Resources
```
Error message → Use Bash-Cheat-Sheet.md
Concept confusion → Read 35-Day-DevOps-Bash-Mastery.md
Syntax issues → Use bash -n script.sh to check
Debugging → Use bash -x script.sh to see execution
```

### Step 2: Common Issues & Solutions
```
"Permission denied" 
→ chmod +x script.sh

"Command not found"
→ Make sure you're using ./script.sh (with ./)

"Variable not expanding"
→ Use $variable not $"variable"

"Script runs wrong"
→ Add set -x after #!/bin/bash to debug
```

### Step 3: Problem-Solving Approach
```
1. Read the error message carefully
2. Check syntax with bash -n
3. Test commands individually first
4. Compare with the example in curriculum
5. Try again with debugging enabled
```
