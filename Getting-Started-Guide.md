## 🚨 WHEN YOU GET STUCK

### Problem: Script doesn't run
**Solution:**
1. Check syntax: `bash -n script.sh`
2. Check permissions: `chmod +x script.sh`
3. Check shebang: First line should be `#!/bin/bash`
4. Run with debug: `bash -x script.sh`

### Problem: Don't understand a concept
**Solution:**
1. Reread that section slowly
2. Type the example exactly
3. Test it in your terminal
4. Modify the example slightly
5. If still stuck, sleep on it and try tomorrow

### Problem: Script runs but gives wrong output
**Solution:**
1. Add `echo` statements to debug
2. Check variable values: `echo $var`
3. Trace execution: Add `set -x` after shebang
4. Compare with provided solution
5. Test with different input

### Problem: Feeling overwhelmed
**Solution:**
1. This is normal! Pace yourself
2. You can stretch to 50 days if needed
3. Review previous days
4. Do easier exercises first
5. Remember: consistency beats speed
